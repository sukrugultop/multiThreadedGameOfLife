
Run class is the main class.
You should give [M] [N] [maxGenerations] or [M] [N] [maxGenerations] [input] as arguments.
M : #rows of input
N : #columns of input
maxGenerations : # lifecycles
input : optional input array. 
It generates output.txt file and it contains each generation method and in case of 3 arguments are given it writes input file in the output file. It also writes that output to console.
